link to github

发表于2021-12-08|更新于2021-12-08
|阅读量:

1.远端：github

1.1创建仓库

注意：路径格式：你的用户名.github.io

2.本地：blog 文件夹下

2.1 安装插件

cnpm install –save hexo-deployer-git

2.2 设置 _config.yml 文件

vscode中 打开这个文件，在这个文件最下面找到Deployment

2.2.1 改type: ‘git’

2.2.2 加repo

repo: ‘仓库地址’

2.2.3 加branch

branch: ‘master’

3.部署到远端

hexo d

输入用户名、密码，然后授权
